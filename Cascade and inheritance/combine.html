CSS Variables with the Cascade, Inheritance, & Specificity

  • Green from :nth-of-type()

CSS Variable value goes to the most specific (and last rule when rules with equal specificity exist). How properties typically inherit, however, also still apply.

So even if you have a more specific rule set the value on an ancestor, if there is a rule that sets the value for the given element, the given element's rule will be applied.
